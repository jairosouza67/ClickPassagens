1. VISÃO GERAL DO PROJETO
================================================================================

OBJETIVO:
Desenvolver uma plataforma web de busca e emissão de passagens aéreas utilizando 
milhas, baseada no modelo do Busca Milhas, mas com identidade visual própria e 
funcionalidades aprimoradas.

PROPOSTA DE VALOR:
- Centralizar busca de passagens por milhas em múltiplas companhias aéreas
- Comparar preços entre milhas vs. dinheiro em tempo real
- Gerar orçamentos personalizados para agentes de viagem
- Sistema de comissões e precificação flexível
- Interface intuitiva para usuários premium e básicos

DIFERENCIAL COMPETITIVO:
- Design moderno e responsivo
- Melhor UX/UI que o site base
- Funcionalidades premium mais acessíveis
- Sistema de indicação e cashback
- Suporte multilíngue

3. ARQUITETURA TÉCNICA RECOMENDADA
================================================================================

3.1 FRONTEND:
- Framework: React.js ou Next.js
- UI Library: Material-UI ou Ant Design

6. REGRAS DE NEGÓCIO
================================================================================

6.1 SISTEMA DE CONSULTAS:
- Consulta = busca em uma companhia específica
- Exemplo: busca Gol + Azul = 2 consultas
- Consultas não utilizadas não acumulam
- Reset mensal do limite de consultas

6.2 PRECIFICAÇÃO:
- Milheiro configurável por companhia
- Faixas de preço por volume de milhas
- Margem de lucro configurável
- Preços dinâmicos baseados na demanda

6.3 COMISSÕES:
- Comissão mínima: R$5 por emissão
- Comissão máxima: 15% do valor
- Pagamento mensal (até dia 15)
- Retenção de 30 dias para estornos

6.4 PROGRAMA DE INDICAÇÃO:
- Comissão: 20% do primeiro pagamento do indicado
- Limite: 10 indicações por mês (usuário básico)
- Sem limite para usuários premium
- Pagamento junto com comissões regulares

6.5 CASHBACK:
- Percentual: 1% para básico, 2% para premium
- Valor mínimo para resgate: R$50
- Prazo para resgate: 90 dias
- Conversão: 1 real = 1000 pontos
